Zum Spiel

Diese Website

Um das WebSocket-Spiel herum ist diese Website entstanden. Sie läuft ebenfalls über Deno, wobei Deno hier nur das Routing übernimmt und als klassischer WebServer fungiert. nginx nutze ich als Reverseproxy, der HTTP- und HTTPS-Anfrage animmt und intern an den Port des Deno-Servers weiterleitet. Let's Encrypt hat das TSL-Zertifikat für den nginx-server asugestellt. Den Source-Code der Website findet man auf meiner Github-Seite

HTML/CSS JavaScript Deno nginx Linux
Details

ein Linux/Debian-Server

Dieser DenoServer bzw. nginx laufen auf einer virtuellen Maschine, welche wiederum auch auf einem Linuxsystem läuft. Über DynDNS wird dem Server eine feste Domain zugewiesen. Somit ist er ganz leicht unter der Domain josua-lucas.de erreichbar.
